生活利器

目前博客用的我新的ID，我只是想找个地方写写我的情感摆了，但还是会被人认出来。

理清了杂乱无章的生活后，单纯的分享下。

介绍一下你自己和所做的工作。

我是一名不合格的黑客，纯野路子出生，无基础、无系统性学习过，现在某公司攻防团队普通员工。

职业生涯中大部分时间在日站，获取犯罪人员信息、数据、服务器权限，以拿权限为目的的渗透。

在努力成为一名真正的黑客而努力着。

你职业生涯的转折点是什么？

将爱好与工作结合在了一起，野路子学习过程，从工厂流水线到修电脑到运维，我命由我不由天最后获得了一份理想的职业。

你都在使用哪些硬件？

我对手机没什么概念，能用就行，最好能挡子弹那种。

电脑：MacBook Pro 2015 + 台式机

平板：iPad mini，kindle

摄像：GoPro HERO7

云主机：科学上网ShadowSocks，L2TP VPN

安全硬件：拓实300M网卡，NFC，橡皮鸭，BadUSB

软件呢？

操作系统：Windows10专业版，Win7企业版，Win2008r2，Kali，OS X，Vmware Fusion，Parallels Desktop

黑客工具：各类网上公开工具

IDE：PyCharm，Visual Studio 2017

视频：Final Cut Pro，ApowerREC

笔记应用：Mweb，印象笔记，Typora

其它：zoommy（博客图片全来源于此），1Password（密码管理）,Docker…

你最理想的工作环境是什么？

一张大木桌，灯光暖色，靠着窗，一个显示屏，一台笔记本，一壶咖啡。

你平时获得工作灵感的方式有哪些？

看任务难度情况，平常工作偶而会抽抽烟，但不会烟很浓的烟薄荷味有爆珠。

如果只是提神，我会服用药物有时候需要非常大精神状态下会服用莫达非尼，其它情况用阿莫达，这两种药是我在查阅各种论文后偶而才会服用，副作用就是把第二天的精神透支。

工作效率最高是在深夜。
