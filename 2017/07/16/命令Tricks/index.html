这本手册里面有大量各种实用的命令，本想一一记录下来有用的，但太多了记录了一点点就不想记录了，我觉得还是可以值得打印出来慢慢翻阅的一本手册。在一些实战中非常有用。

Linux Network Commands

Command Description
watch ss -tp 网络连接
lsof -i 建立连接
iwlist int scan 无线wifi扫描
ip xfrm state list 打印vpn密钥
echo “1” /proc/sys/net/ipv4/ip_forward 打开转发
echo “nameserver x.x.x.x” /etc/resolv.conf 添回DNS

Linux Utility Commands

Command Description
rdesktip Remote
scp /tmp/file user@x.x.x.x:/tmp/file Put File
scp user@x.x.x.x:/tmp/file /tmp/file Get File
rmuser uname Remove user
script -t 2>time.file -a output.file Record Shell
scriptreplay time.file output.file 播放
apropos -k uname 查找相关命令

Record:
script -t 2>demo.time -a demo.his
scriptreplay demo.time demo.his

利用script记录某人行为:
script -t -f -q 2>$USER.time -a $USER.his

查看记录每个人对线上服务器的操作
