一星期实战总结（五）

这个星期主要还是做《一星期实战总结》(四)的工作，本以为获取到了文件服务器查找到有关的文件可以结束了，过了一个星期之后发来了一堆要求，直接给了word文档，文档各种要求包括某序列号合同等。没办法超起刀继续搞。续上次已经拿到了域控，但没有真正的去搞想着其它机器没有我想要的机器。

之前在服务器中发现公司密码不过是经过加密的，跑了6位路生成的密码字典没法解出来就放弃了，因为我给机器是种了2种不同的远控，所以一直没有被发现，连一些国外的杀软都没有查杀出来，之后我就锁定了内网的某系统、以及Exchange、域服务器,其中域服务器分为两台，一台windows2003、另外一台Windows2008。

跳板 -> 域服务器 -> 抓取hashdump -> 获取各类系统中的用户和密码 -> 域用户可以登录Exchange -> 域用户也就是Exchange用户密码。

这个星期的总结写的比较少，下次每天都记录下来吧，不然一个星期过去了，很多细节都想不起来。

最近爆出来的系统级别的0day，这里来说说我的感受，国外一支小团队都拥有这么多0day，每个组织都有不同的攻击风格，这里只是冰山的一角，像我们这类人又没有0day，也没有任何的支持和技术的分享交流什么的，一个团队也很难走，那么一个人就更难走了。

在这方向主要锻炼一个人的实战能力，如果长久呆下去或许也就这样子吧，没有任何的前景发展。
