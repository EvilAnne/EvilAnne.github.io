塔建开发环境

Virtualenv

virtualenv就是用来为一个应用创建一套隔离的Python运行环境。

安装:
pip3 install virtualenv

创建虚拟环境:
virtualenv ./blogproject_env

激活:
source ./blogproject_env/bin/activate

安装Django:
pip3 install django

如果想指定安装某个版本：
pip3 install django=1.11.11

测试是否安装成功：

创建Django项目

创建：
django-admin startproject blogprojecta

blogprojecta目录下存Django配置文件

Django APP

创建：
python manage.py startapp blog

之前通过django-admin startapp blog来创建后面会遇到各种各样的问题这里不推荐使用此方法。

settings.py文件找到INSTALLED_APPS将blog添加进去

创建数据库模型

Django 为我们提供了一套 ORM（Object Relational Mapping）系统它有自己一套规定的语法来生成表结构。

Category就是一个标准的 Python 类，它继承了 models.Model 类，类名为 Category 。Category 类有一个属性 name，它是 models.CharField 的一个实例。

CharField(max_length=100)是字符型 max_length指定其最大长度
TextField() 文本类型
DateTimeField()存储时间字段
CharField(max_length=200, blank=True)指定字符型长度100可以为空
ForeignKey(Category)即一对多的关联关系
ManyToManyField(Tag, blank=True)多对多的关联关系。
